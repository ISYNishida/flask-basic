# Flask 入門サンプル（Step1 / Step2 / Step3）

このリポジトリは、Python の Web フレームワーク **Flask** を使った  
基本的な Web アプリケーション構成を学ぶためのサンプルです。

CGI での Web プログラムを出発点として、

- フレームワークを使うと何が簡単になるのか
- Web の基本的な入出力（リクエスト／レスポンス）
- 処理と表示の分離（テンプレート）

といった点を、段階的に確認できる構成にしています。

## ディレクトリ構成
step1/ HTML を Python から直接返す最小の Flask アプリ
step2/ テンプレート（HTML）を分離した構成
step3/ データ保存・検索を含む簡単な Web アプリ

各 step は独立しており、上から順に試すことで  
Flask の基本的な考え方を理解できるようになっています。

## 対象読者

- Python の基礎文法を一通り触ったことがある方
- CGI や Web の仕組みに興味がある方
- フレームワークを「使う前に仕組みを知りたい」方

実務向けの完成形というより、  
**学習用・理解重視のサンプル**です。
